# ポートフォリオ
就職活動用に自身のポートフォリオとして主にRuby on Railsを使用してWebアプリケーションを作成しました。
## アプリケーション名  
InMyCloset  
## 概要
クローゼット管理, 共有アプリケーション
## 機能一覧
* フォロー機能
* いいね機能
* アプリ上に洋服を保存, 編集
* ブランド検索, カテゴリー検索, ユーザー検索が可能
## インフラ
AWS
## Rubyのバージョン
ruby-2.5.7
## Railsのバージョン
rails-5.2.4
## 使用しているGem
* font-awesome-rails(5.4.1)
* refile
* refile-mini_magick
* bootstrap-sass
* jquery-rails
* devise
